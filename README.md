这是一个用MEAN开发的博客系统。
如果不了解MEAN：请查看http://mean.io/。
本开源项目，依赖node、NPM、mongodb环境请自行到官网下载安装。

MongoDB的配置在db.json中
npm包依赖在package.json中

路由用angularJS来控制。
后台仅提供数据接口和业务处理接口。

启动该项目，需要执行 node app.js, 访问localhost:3000即可。
